常见问题

GFM是什么？支持哪些功能？ ▼
GFM (GitHub Flavored Markdown) 是GitHub扩展的Markdown语法，包括：表格、任务列表（复选框）、删除线、自动链接、代码块语言标识等。本工具默认启用GFM扩展。
如何创建表格？ ▼
使用竖线和连字符创建表格：
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| 内容1 | 内容2 | 内容3 |
第一行是表头，第二行定义对齐方式（:--左对齐，:--:居中，--:右对齐）。
如何创建任务列表？ ▼
使用 [ ] 和 [x] 创建复选框：
- [x] 已完成任务
- [ ] 未完成任务
勾选GFM任务列表选项即可渲染为可交互的复选框样式。
代码块如何指定语言？ ▼
在开头的 ``` 后指定语言：
```javascript
const hello = 'world';
```
支持 javascript, python, html, css, json, bash 等常用语言。
生成的HTML可以直接使用吗？ ▼
生成的HTML只包含内容部分（body内容），不包含样式。如需完整网页，可以复制HTML源码后自行添加 <html>、<head>、<style> 等结构，或直接嵌入到已有网页中。
